Установите NodeJS

Открываем командную строку и переходим в каталог проект
cd + путь к проекту
Например, cd D:\Projects\WebSite\Psyho

Далее, перед первым запуском приложения необходимо установить библиотеки:
выполняйте поочередно
1. npm install express
2. npm install cors
3. npm install sqlite3

Далее для запуска приложения вводим
node app.js

Открываем браузер и переходим по ссылке 
95.213.224.23:3000
